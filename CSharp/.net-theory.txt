
.NET is a free, open-source software development platform created by Microsoft for building:

Web Applications
Web APIs
Desktop Applications
Mobile Apps
Cloud Applications

Enterprise Software

The most common language used with .NET is C# (C-Sharp).

Think of .NET as a complete ecosystem that provides:

Programming languages (C#, F#, VB.NET)
Runtime (CLR)
Libraries
Tools for development
