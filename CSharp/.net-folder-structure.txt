| MERN              | ASP.NET Core MVC            | Purpose                 |
| ----------------- | --------------------------- | ----------------------- |
| server.js/app.js  | Program.cs                  | Application Entry Point |
| Routes            | Controllers                 | Handle Requests         |
| Models (Mongoose) | Models + Entity Framework   | Database Structure      |
| React Components  | Views (.cshtml)             | UI Rendering            |
| Middleware        | Middleware                  | Request Pipeline        |
| MongoDB           | SQL Server/MySQL/PostgreSQL | Database                |
| Services          | Services                    | Business Logic          |
| Public Folder     | wwwroot                     | Static Files            |


1 ) Program.cs

   in MERN
   const express = require("express");
   const app = express();
   
   app.listen(3000);
   ------------------
   in ASP.NET
   var builder = WebApplication.CreateBuilder(args);
   
   builder.Services.AddControllersWithViews();
   
   var app = builder.Build();
   
   app.MapControllerRoute(
       name: "default",
       pattern: "{controller=Home}/{action=Index}/{id?}");
   
   app.Run();

Responsibility : 
   Start application
   Configure services
   Configure routes
   Dependency Injection


2 ) Controllers     

   in JS 
   router.get("/users", getUsers);
   -----
   in ASP.NET 
   public class UserController : Controller
   {
       public IActionResult GetUsers()
       {
           return View();
       }
   }
   
   Responsibility = 
     Receive HTTP Request
     Call Service
     Return View or JSON

3 ) Models   
    in MERN - mongoose 

    const UserSchema = new mongoose.Schema({
        name:String,
        email:String
    });
    -----
    in ASP.NET
    public class User
    {
        public int Id { get; set; }
    
        public string Name { get; set; }
    
        public string Email { get; set; }
    }
  Responsibility = 
    Database Entity
    Data Transfer
    Validation

5 ) Data Folder :

9 )  Migrations : 
     in MERN
     
     Mongo creates collections automatically.
     
     in ASP.NET - command below 

       --Add-Migration InitialCreate
       --Update-Database
     
     Generates: 
     Migrations/
     
     it Contains SQL schema changes.
     
     ex. Version control for Database
